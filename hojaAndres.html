HABILIDADES


Programación en Java

Es crear programas usando un lenguaje llamado Java. Este lenguaje se usa para hacer desde aplicaciones para celulares hasta sistemas complejos como los de bancos o parqueaderos.

Programación en HTML y CSS

Es diseñar páginas web. HTML sirve para crear la estructura de la página (dónde van las imágenes, textos, botones, etc.). CSS se usa para darle estilo (colores, tamaños, letras bonitas, etc.).

Programación de bases de datos SQL

Es organizar y manejar datos en un sistema. Por ejemplo, puedes guardar información sobre clientes, productos o vehículos en un lugar seguro llamado "base de datos" y usar SQL para buscar o modificar esos datos.

Mantenimiento de equipos

Es revisar y arreglar computadoras, impresoras y otros dispositivos para asegurarte de que funcionen bien. Esto incluye limpiar, reparar o actualizar componentes cuando sea necesario.
